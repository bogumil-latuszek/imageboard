# Imageboard Project - VS Code Setup

## Prerequisites
- .NET sdk
- SQLite
- VS Code
- VS Code Extensions:
  - C#
  - C# Dev Kit
  - .NET Install Tool

## First-Time Setup
1. Clone the repository
2. Open folder in VS Code
3. Restore dependencies:
```bash
dotnet restore
```
4. Update Database:
```bash
dotnet ef database update
```

## Build and Run the Project
```bash
dotnet run

```
